AWS Cloud Infrastructure — EC2 (lifecycle, instance types), Auto Scaling, horizontal vs vertical scaling, Elastic IP, and AMI basics.

Docker & Containerization — why Docker exists, containerization concept, Docker architecture, images vs containers, Docker Engine, container lifecycle, and the full Dockerfile (FROM, RUN, COPY, WORKDIR, CMD, ENTRYPOINT, layer caching).

AWS Lambda & Serverless — serverless concept, event-driven architecture, Lambda lifecycle, stateless execution, cold start problem, memory allocation, timeout and retry behavior, scaling, cost optimization, and when not to use Lambda.

SQS & Message Queues — producer-consumer model, async processing, visibility timeout, Dead Letter Queue (DLQ), retry mechanisms, and queue durability.

Database Scaling & Indexing — why indexes improve performance, B-Tree basics, clustered vs non-clustered indexes, composite indexes, index tradeoffs, over-indexing, query optimization, and read replicas.

Backend Security — JWT basics, session vs token authentication, API key authentication, and firewall/security group concepts.

Notification Systems — email/SMS/push notifications, queue-based notification systems, retry mechanisms, failure handling, bulk notifications, fanout, and throttling.

Reliability & Failure Engineering — retry mechanisms, exponential backoff, retry storms, DLQ handling, high availability, redundancy, and failover.

Comparative Theory — Synchronous vs Asynchronous processing, Docker vs VM, SQL vs Redis, Horizontal vs Vertical scaling.
